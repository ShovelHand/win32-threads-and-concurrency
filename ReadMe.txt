I made this program to gain some experience with OS principles in the Windows environment 
that I am already familiar with in Linux/Unix.
This program demonstrates multithreading, as well as attention to mutual exclusion and 
synchronization concerns.
This program is an implementation of a classic  U.C. Berkeley concurrency problem which can be 
read about at http://www.greenteapress.com/semaphores/downey05semaphores.pdf, p.155
